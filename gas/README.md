# 🐱 白猫シェフ TikTok動画クリエイター

Google Apps Script (GAS) で白猫シェフのAI料理動画ストーリーボードを自動生成するスクリプトです。

---

## 📋 生成される成果物

| 成果物 | 内容 |
|--------|------|
| **Google Slides ストーリーボード** | 全シーンの画像・ナレーション・タイミングをまとめた絵コンテ |
| **ナレーション台本（Google Doc）** | にゃ〜入りのシーン別セリフ・SE指示・音声制作ガイド |
| **DALL-E 3 生成画像** | 各シーンの白猫シェフ画像（9:16縦型、Google Drive保存） |

---

## 🎬 動画仕様

- **フォーマット**: 縦型 9:16（1080×1920px） / TikTok向け
- **合計時間**: 60秒以内
- **各シーン**: 最大5秒
- **シーン構成**:
  1. オープニング（にゃ〜！）
  2. 材料紹介
  3. 調理ステップ × 最大10シーン
  4. エンディング（完成にゃ〜！）

---

## 🚀 セットアップ手順

### 1. Google Apps Script プロジェクトを作成

1. [script.google.com](https://script.google.com) にアクセス
2. 「新しいプロジェクト」をクリック
3. プロジェクト名を `白猫シェフTikTok` に変更

### 2. ファイルをコピー

スクリプトエディタで以下の `.gs` ファイルを作成し、それぞれコードをコピー＆ペーストしてください：

| ファイル名 | 役割 |
|-----------|------|
| `Config.gs` | 設定・定数 |
| `Utils.gs` | ユーティリティ関数 |
| `Recipe.gs` | レシピ生成（GPT-4o） |
| `SceneBuilder.gs` | シーン構成 |
| `ImageGenerator.gs` | 画像生成（DALL-E 3） |
| `NarrationScript.gs` | ナレーション台本 |
| `Storyboard.gs` | Google Slides作成 |
| `Code.gs` | メイン処理・テスト関数 |
| `WebApp.gs` | Webアプリ（任意） |

`appsscript.json` の内容も「プロジェクトの設定」>「appsscript.json を表示」でコピーしてください。

### 3. OpenAI APIキーを設定

1. スクリプトエディタ > **プロジェクトの設定** > **スクリプトプロパティ**
2. 「プロパティを追加」をクリック
3. プロパティ名: `OPENAI_API_KEY`　/　値: あなたのOpenAI APIキー

> ⚠️ APIキーはコードに直接書かないでください。スクリプトプロパティに安全に保管されます。

### 4. 実行

スクリプトエディタで `testOmurice` を選択して ▶ 実行ボタンをクリック！

## 📖 使い方

### スクリプトエディタから実行

```javascript
// オムライスで動画素材を作成
testOmurice();

// 任意の料理名で作成
createCatCookingVideo('唐揚げ');
```

### Webアプリとして公開する場合（任意）

1. スクリプトエディタ > **デプロイ** > **新しいデプロイ**
2. 種類: **ウェブアプリ**
3. 次のユーザーとして実行: **自分**
4. アクセスできるユーザー: **全員**
5. デプロイして表示されたURLにアクセス

## ❓ トラブルシューティング

**Q: 画像生成がエラーになる**
→ OpenAI APIの残高を確認してください。DALL-E 3 は従量課金です。

**Q: スライドに画像が表示されない**
→ Drive APIの権限を確認してください。初回実行時は権限の許可が必要です。

**Q: 実行時間が長すぎる**
→ GASの実行制限（6分）に引っかかる場合があります。ステップ数を減らすか、シーンを分割して実行してください。
