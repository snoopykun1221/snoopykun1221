## 最初にやること：「今の仕事フロー」に差し込む場所を1つ決める

ChatGPTを「全部に使おう」とするのが失敗のパターン。

最初に決めること：**自分の仕事の中で「毎日必ず発生する」タスクを1つ選んで、そこにだけChatGPTを使う。**

自分が選んだのは「日報の作成」だった。

以前は日報を15〜20分かけて書いてた（何をやったか思い出しながら文章にする作業）。これをChatGPTに任せるようにした。

**やり方：**
今日やったことをメモ（箇条書き、雑でいい）をChatGPTに投げて「これを日報の形式に整えて」と送る。出てきた文章を少し修正して送信する。

5分で終わるようになった。

毎日使うので、1週間後には「日報→ChatGPTで書く」が自動化されてた。使う習慣ができると、自然と他の用途にも使い始めた。

## 仕事の種類別・実際の使い方

3ヶ月使い込んで、効果が高かった用途を書く。

### 1. 文書作成・メール

**一番使う用途。**

自分が書きたい内容を箇条書きで投げる→ChatGPTが文章化→自分が修正。

「0から書く」より「直す」の方が圧倒的に速い。ライティングが苦手な人ほど効果が高い。

**プロンプトの型：**
```
以下の内容を[メール/報告書/提案書]の形式に書いてください。

受け手：[誰に対して]
目的：[何を伝えたい・何をしてほしい]
内容：
・[箇条書きで内容を書く]
・[箇条書きで内容を書く]

トーン：[丁寧/フランク/簡潔に]
```

修正依頼も上手く使う：
```
この文章の[この部分]を[こう変えて]ください。他は変えないでください。
```

### 2. 情報収集・調査

「○○について教えて」だと曖昧な答えが返ってくる。

**効果的な聞き方：**
```
[テーマ]について、以下の観点で教えてください。
・[知りたい観点1]
・[知りたい観点2]
・[知りたい観点3]

私の前提知識：[どのくらい知っているか]
用途：[何のために知りたいか]
```

前提知識と用途を伝えると、説明のレベルが合ったものが返ってくる。

ただし、ChatGPTは事実確認に弱い（もっともらしい誤情報を出すことがある）。**調査結果は必ず一次情報で確認する。** これは鉄則。

### 3. アイデア出し・壁打ち相手

これが想定外に役立った。

新しい企画や提案を考えるとき、「自分の考えを整理する」のに使う。

```
[課題]について考えています。
今考えているアプローチは[自分の案]です。
このアプローチの問題点、見落としているリスク、別のアプローチを教えてください。
```

人に相談する前に、自分の考えを整理するステップとして使う。上司や同僚への相談の質が上がった。

### 4. 会議・打ち合わせの準備

打ち合わせの前に：
```
明日[○○についての会議]があります。
私の役割：[立場]
議題：[概要]
事前に準備しておくべき質問と確認事項を教えてください。
```

これをやるようになって、会議での発言の質が変わった。

## 「使い続けるための」具体的な設定

ツールを習慣化するには、使うまでの摩擦を下げることが重要。

**やったこと：**
- ブラウザのホームページをChatGPT（chat.openai.com）に設定した
- スマホのホーム画面1ページ目にアプリを置いた
- よく使うプロンプトをChatGPTの「カスタム指示」に設定した

**カスタム指示の設定（Settings → Personalization → Custom instructions）：**

「私について：」の欄に自分の仕事の概要・よく使う用途を書いておくと、毎回説明しなくていい。

例：
```
私は中小企業のマーケティング担当者です。
主な業務：メール作成、資料作成、SNS運用。
好みのスタイル：箇条書きよりも文章、丁寧すぎない言葉使い、具体例を含めること。
```

## 生産性への影響（3ヶ月後の変化）

時間で測った変化：

- 日報作成：20分→5分（週5日で週1時間15分削減）
- メール作成（長文）：30分→10分
- 資料の初稿作成：3時間→1時間
- 会議準備：0分→15分（質が上がったので全体の会議時間が短縮）

「ChatGPTを使う時間」を加えても、週に5〜6時間の純粋な時間削減になった。

その時間を何に使ったか：新しい企画の立案と、副業の記事執筆。

## よくある失敗と対策

**「出てきたものをそのまま使う」**
ChatGPTの出力は「たたき台」として使う。そのまま使うと、一本調子で個性のない文章になる。自分の言葉で直す作業が必要。

**「プロンプトが雑になる」**
慣れてくると「○○について教えて」みたいな雑な聞き方に戻ってしまう。出力の質が下がったと感じたら、プロンプトの詳細度を上げることを確認する。

**「事実確認をしない」**
数字・固有名詞・最新の情報はChatGPTを信じない。確認する習慣は絶対に手放さない。

## まとめ

ChatGPTで生産性を上げるのに、特別な知識は要らない。

「毎日必ずやる仕事」に1つだけ組み込む。使い続けることで精度が上がる（自分のプロンプトの精度が上がる）。そこから自然に用途が広がる。

「すごい使い方」を探すより、「毎日使い続ける場所を1つ作る」方がずっと大事。
